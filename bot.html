Bot Discord Musical — Team NoxxWorks

Un bot musical puissant construit en Python pour usage interne — YouTube, Spotify & Deezer, radio intégrée, IA d'assistance, système de sécurité et modération professionnel. Conçu pour rester en ligne 24/7 et donner le meilleur contrôle aux modérateurs.

Bot privé — accès restreint Ouvrir le panneau
  • • Lecture YouTube / Spotify / Deezer
  • • Radio (Radio‑Browser) & pré‑configs France
  • • Autocompletion intelligente & panels interactifs

Fonctionnalités principales

Résumé rapide des modules intégrés.

Lecture & File d'attente

YouTube (yt-dlp), conversion Spotify/Deezer -> YouTube, queue, shuffle, loop, autoplay et 24/7.

Radio

Intégration Radio‑Browser (30k+ stations). Panels dédiés, recherches par pays, favoris FR préconfigurés.

Assistant IA

Assistant (Gemini-like) contextualisé au serveur, multi‑tour, mémoires et configuration persistante.

Sécurité & Modération

Anti‑raid / anti‑spam / sanctions persistantes / logs détaillés / systeme de sanctions avec IDs.

Dev Tools & Logs

Commandes développeur restreintes, système de logs (embed), diagnostics et analytics.

Autocompletion

Autocompletion intelligente pour musique (Deezer/YouTube) et radios (API live).

Tech & dépendances

  • Langage : Python 3.11+
  • Libs : discord.py[voice], yt-dlp, spotipy, deezer-python, PyNaCl, aiohttp
  • Stockage : SQLite / PostgreSQL (optionnel)

Le projet est pensé pour être modulaire : modules séparés pour musique, radio, IA, sécurité et modération afin de faciliter les mises à jour sans interruption.
